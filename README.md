# AI Resume Analyzer

AI-powered ATS Resume Analysis platform built using **Next.js**, **FastAPI**, and **Google Gemini**.

The application evaluates resumes against modern ATS (Applicant Tracking System) standards and provides intelligent feedback to improve hiring success rates.

## Features

### ATS Compatibility Analysis

Evaluate how effectively a resume can pass through Applicant Tracking Systems.

### AI Resume Summarization

Generate concise professional summaries using Gemini.

### Keyword Extraction Engine

Identify important technical and domain-specific keywords.

### Strength & Weakness Detection

Analyze resume structure, content quality, and presentation.

### Intelligent Recommendations

Receive actionable suggestions to improve ATS performance and recruiter visibility.

### Professional Feedback Report

Generate a structured analysis report with optimization insights.
